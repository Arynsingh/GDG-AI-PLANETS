About the Model

🧠

Pix2Pix GAN

A conditional GAN with an encoder–decoder generator and PatchGAN discriminator, trained end-to-end on paired compound images.

📊

Training

Trained for 10 epochs on 1,800 compound images (600 per class) at 256×256 resolution using Adam optimizer (lr=2×10⁻⁴).

🔬

Application

Generated images can be used for molecular analysis, data augmentation, and exploring structural variations of known compounds.

⚡

Tech Stack

Built with TensorFlow 2.x, Flask, and NumPy. The web interface is served via a lightweight Python backend with real-time inference.
